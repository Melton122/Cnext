=====================================
  Cnext v9.0 - Installation Guide
=====================================

QUICK START
-----------

1. Extract this ZIP to a folder (e.g., C:\Cnext)

2. Add the bin folder to your PATH:
   - Open System Properties > Advanced > Environment Variables
   - Under System Variables, find "Path" and click "Edit"
   - Click "New" and add: C:\Cnext\bin
   - Click "OK" to save

3. Open a new command prompt and verify:
   cnext --version

4. Create your first program:
   - Create a file called hello.cn with:
     main {
         printin("Hello, World!")
     }

5. Run it:
   cnext run hello.cn


COMMANDS
--------

  cnext run <file.cn>       Compile and run a Cnext file
  cnext build <file.cn>     Compile to an executable
  cnext test [file.cn]      Run tests
  cnext new <project>       Create a new project
  cnext fmt <file.cn>       Format a Cnext file
  cnext lint <file.cn>      Lint a Cnext file
  cnext repl                Start interactive REPL
  cnext doc <file.cn>       Generate documentation
  cnext install [package]   Install packages
  cnext version             Show version
  cnext help                Show help


REQUIREMENTS
------------

- GCC compiler (MinGW or MSYS2) in your PATH
- Windows 10 or later


INCLUDED
--------

  bin/          Compiler executable
  include/      Runtime headers and standard library
  examples/     Example programs
  tests/        Test files
  docs/         Documentation


TROUBLESHOOTING
---------------

If "gcc not found":
  - Install MinGW: https://mingw-w64.org/
  - Or install MSYS2: https://www.msys2.org/
  - Add the bin folder to your PATH

If "cnext not found":
  - Make sure you added the bin folder to PATH
  - Or use the full path: C:\Cnext\bin\cnext.exe
